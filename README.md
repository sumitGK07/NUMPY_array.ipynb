NumPy Arrays in Python 🐍

This assignment focuses on learning and implementing the fundamental concepts of **NumPy**, one of the most powerful Python libraries for numerical computing and data analysis. 🚀

Throughout this assignment, I explored how to create, manipulate, and analyze **1D, 2D, and 3D arrays** using various NumPy functions. The programs cover array creation, reshaping, indexing, slicing, statistical operations, arithmetic operations, matrix multiplication, and advanced indexing techniques. 📊

### 📚 Topics Covered

* ✅ Creating NumPy arrays using different methods
* 📏 Array properties (Shape, Size, Data Type)
* 🔢 Statistical methods (Minimum, Maximum, Sum, Mean, Standard Deviation)
* 🔄 Reshaping arrays into different dimensions
* 🎯 1D & 2D Array Indexing and Slicing
* 🔍 Boolean and Advanced Indexing
* ➕ Arithmetic Operations on Arrays
* ✖️ Matrix Multiplication
* 📈 Random Number Generation using NumPy
* 🎓 Student Performance Analysis Mini Project

### 💡 Learning Outcomes

By completing this assignment, I gained practical knowledge of:

* Writing efficient programs using NumPy arrays.
* Performing mathematical and statistical computations with built-in NumPy functions.
* Manipulating array dimensions using `reshape()`.
* Accessing and modifying array elements using indexing and slicing.
* Applying boolean indexing and conditional operations.
* Performing element-wise and matrix operations.
* Solving real-world data analysis problems using NumPy.
